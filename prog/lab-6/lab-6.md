# Лабораторная работа №6. Использование шаблона «Наблюдатель»

## Задание

Необходимо создать программу на языке Python, которая использует паттерн проектирования "Наблюдатель" для отслеживания изменений курсов валют через API Центробанка РФ. Программа должна запрашивать курсы валют и уведомлять зарегистрированных наблюдателей о изменении курсов в реальном времени или через заданные интервалы времени.
Структура реализованного задания должна представлять

Объект — веб-сервер Flask или FastAPI, Tornado.
Наблюдатели - клиенты, представляющие HTML-страницы, связывающиеся с объектом с помощью веб-сокетов. На странице должен отображаться идентификатор клиента.

## Решение:

Учебный проект, демонстрирующий паттерн проектирования **«Наблюдатель» (Observer)**
на примере отслеживания курсов валют через API Центробанка РФ.

* **Объект (Subject)** — веб-сервер на **FastAPI**, который опрашивает API ЦБ,
  хранит последний снимок курсов и рассылает уведомления об изменениях.
* **Наблюдатели (Observers)** — HTML-страницы клиентов, подключённые по
  **WebSocket**. На каждой странице отображается её уникальный идентификатор.
  Дополнительно есть серверные наблюдатели-примеры, следящие за конкретными
  валютами (USD, EUR, GBP).

## Аннотации типов

Весь код типизирован (PEP 484/585/604):

* доменные модели — `@dataclass(frozen=True, slots=True)`;
* формы сообщений WebSocket — `TypedDict` (`ServerMessage`, `RateDTO`, `ChangeDTO`);
* источник курсов описан как `Protocol` (`RateProvider`), что позволяет
  подменять реальный клиент имитатором без изменения кода сервера.
